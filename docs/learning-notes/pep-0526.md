# PEP 526 (Anotações de Variáveis)

Esta é uma tradução livre e resumida da PEP 526.  
Para o texto oficial, consulte: https://peps.python.org/pep-0526/

---

## Objetivo

Permitir anotar tipos diretamente em variáveis, tornando o código mais claro e ajudando ferramentas de análise.

---

## Ideia principal

Além de anotar parâmetros e retornos, você pode anotar variáveis comuns:

```python
count: int = 0
name: str = "Rogerio"
is_active: bool = True
```

Isso não muda o runtime, apenas documenta intenção.

---

## Variáveis sem valor inicial

```python
total: float
```

Aqui você declara a intenção do tipo mesmo antes de definir o valor.

---

## Coleções anotadas

```python
from typing import List, Dict

scores: List[int] = [10, 20, 30]
labels: Dict[str, int] = {"high": 3, "low": 1}
```

---

## Benefícios

- Melhora a leitura para humanos.
- Ajuda ferramentas como mypy e IDEs.
- Facilita a manutenção do código com menos ambiguidades.
